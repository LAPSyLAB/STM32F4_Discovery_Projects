 Podana so osnovna navodila za uporabo aplikacije.
 Njena vsebina in delovanje sta bolj podrobno opisana v readme.txt.
 Primer je uvozen iz zbirke primerov STM32Cube_F4 distribucije.
 
Postopek priprave in uporabe projekta :
1. CubeIDe :
	File->Import-> Import STM32Cube Example
	izbereš STM32F407 Discovery board in "Audio_playback_and_record" example
	
2. Priprava STM32F4-DISCO boarda
    - Priklop USB kljuca na USB Micro prikljucek
    - "audio_sample.wav" iz STM32Cube_F4 zbirke v mapi "Utilities/Media/Audio" se shrani na USB kljuc (root direktorij)
    
3. Prevajanje projekta v CubeIDE
	Project -> Build 
	Run->Debug

4. Uporaba aplikacije
	priklop slušalk na banana prikljucek
	slisi se predvajanje .wav datoteke (blinka modra LED)
	USER tipka preklopi med predvajanjem in snemanjem
	med snemanjem preko mikrofona utripa oranzna LED dioda - snema se v rec.wav datoteko na USB kljucu
